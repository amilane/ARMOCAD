DL length
ARMOCAD

Описание
Программа собирает длины выбранных пользователем линий детализации и суммирует их.

Общие указания по применению
Выберите линии детализации.
Нажмите кнопку данной программы, появится диалоговое окно
с суммарной длиной выбранных линий.

Скриншоты



Пример использования
Некоторые инженерные коммуникации (провода, кабели и пр.) удобно показывать в Revit линиями детализации.
Для подсчёта длин этих линий (т.е. длин проводов, кабелей) создана данная программа.

Ограничения по применению
Программа работает только с линиями детализации - не с линиями модели.

Установка и удаление
Установка программ происходит при запуске инсталлятора

При выпуске новых версий программ и получении нового инсталлятора рекомендуется сначала выполнить деинсталляцию предыдущей версии.
Чтобы удалить вкладку с программами ARMOCAD из Revit и с компьютера нужно зайти: Пуск - Панель управления - Программы и компоненты
найти в списке установленных программ ARMOCAD и удалить его
