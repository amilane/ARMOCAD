ARMOCAD

Описание
Программа заменяет значение выбранного параметра, если он встречается у всех выбранных семейств.

Общие указания по применению
Выберите семейства в совпадающие параметры которых Вы хотите записать значение.
Нажмите кнопку данной программы, появится диалоговое окно.
Выберите нужный параметр из выпадающего списка.
Введите новое значение для выбранного параметра.
Нажмите кнопку "Записать".

Скриншоты



Пример использования
При выборе разных семейств в их свойствах не всегда показываются их совпадающие параметры.
Это особенно актуально для типовых аннотаций. Например при работе в чертёжных видах (рисовании принципиальных схем и пр.)
возникает необходимость записать какое-либо значение в несколько разных типовых аннотаций. Стандартными средствами Revit
это сделать невозможно. Вот тут и пригодится данная программа.
Программа работает также и для одного выбранного семейства. Так что можно просто записывать значения его параметров из окошка данной программы.
Это порой удобнее чем писать его значение в стандартной панели 'Свойства', где иногда слетает то, что пользователь только что написал.

Ограничения по применению
Программа работает только с параметрами, у которых тип данных:
текст, число, целое число, логическое (в этом случае значение параметра в окошке программы должно быть 1 или 0)

Установка и удаление
Установка программ происходит при запуске инсталлятора

При выпуске новых версий программ и получении нового инсталлятора рекомендуется сначала выполнить деинсталляцию предыдущей версии.
Чтобы удалить вкладку с программами ARMOCAD из Revit и с компьютера нужно зайти: Пуск - Панель управления - Программы и компоненты
найти в списке установленных программ ARMOCAD и удалить его
