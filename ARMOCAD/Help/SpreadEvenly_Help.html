SpreadEvenly
ARMOCAD

Описание
Программа автоматически расставляет выбранные семейства в выбранном помещении.

Общие указания по применению
Запустите программу нажав на икноку SpreadEvenly.
Вам будет предложено указать три точки, описывающие помещение в котором должны быть установлены семейства.
Укажите первую точку (в одном из углов помещения), затем вторую точку (следующий угол помещения по той же стене),
и третью точку (следующий угол по порядку). Таким образом программа получит границы помещения и поймёт внутри какого
пространства нужно работать.
Затем появится диалоговое окно в котором Вам необходимо будет указать выбрать категорию семейств,
конкретное семейство экземпляры которого Вы хотите расставить в помещении,
количество рядов, общее количество и высоту установки семейств от пола.
Также можно задать вращение семейств вокруг собственного центра (может пригодится для прямоугольных объектов).
Нажмите "Расставить" и выбранные Вами семейства появятся в выбранном помещении.

Пример использования
Обводим какую-либо комнату на плане этажа, расставляем в ней нужное количество выбранных семейств.

Ограничения по применению
При использовании программы для расстановки светильников:
Для того чтобы расставить светильники с помощью данной программы нужно знать необходимое количество светильников, обеспечивающих
нормируемую освещённость. Сейчас программа не производит светотехнического расчёта, а только принимает количество светильников
указанных пользователем.

При выпуске новых версий программ и получении нового инсталлятора рекомендуется сначала выполнить деинсталляцию предыдущей версии.
Чтобы удалить вкладку с программами ARMOCAD из Revit и с компьютера нужно зайти: Пуск - Панель управления - Программы и компоненты
найти в списке установленных программ ARMOCAD и удалить его
